使用Hexo的基本操作

* 建站的基本操作
    ```node.js
    hexo new post "test" # 会在 source/_posts/ 目录下生成文件 ‘test.md’，打开编辑
    hexo generate        # 生成静态HTML文件到 /public 文件夹中
    hexo server          # 本地运行server服务预览，打开 http://localhost:4000 即可预览你的博客
    ```
